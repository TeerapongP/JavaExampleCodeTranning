คุณได้รับมอบหมายให้สร้างโปรแกรมตะกร้าช้อปปิ้งออนไลน์โดยใช้ภาษา Java  โปรแกรมควรให้ผู้ใช้สามารถเพิ่มสินค้าลงในตะกร้า คำนวณราคารวม นำส่วนลดมาใช้ และแสดงราคาสุดท้ายหลังหักภาษี
เขียนโปรแกรม Java โดยใช้ Map ที่มีคุณสมบัติต่อไปนี้:
กำหนดแผนที่ชื่อ "products" ที่เก็บชื่อสินค้าเป็นคีย์และราคาสินค้าเป็นค่า กำหนดค่าเริ่มต้นให้แผนที่ดังนี้:
"Apple" - 90.0
"Banana" - 30.0
"Orange" - 50.0
"Mango" - 70.0
"Grapes" - 100.0
แสดงสินค้าที่มีอยู่ให้ผู้ใช้ด้วยการวนลูปผ่านแผนที่ "products" และพิมพ์ชื่อและราคาของสินค้าแต่ละรายการ
แสดงข้อความให้ผู้ใช้ป้อนชื่อสินค้าที่ต้องการเพิ่มลงในตะกร้า (หรือป้อน '!' เพื่อออกจากลูป)
ตรวจสอบว่าสินค้าที่ป้อนอยู่ในแผนที่ "products" หรือไม่ หากมีให้เพิ่มสินค้าลงในแผนที่ "cart" โดยใช้ชื่อสินค้าเป็นคีย์และราคาเป็นค่า และพิมพ์ข้อความยืนยันการเพิ่ม
ถ้าสินค้าที่ป้อนไม่มีอยู่ในแผนที่ "products" ให้พิมพ์ข้อความข้อผิดพลาดที่ระบุว่าไม่พบสินค้า และแจ้งให้ผู้ใช้ป้อนชื่อสินค้าที่ถูกต้อง
หลังจากผู้ใช้เพิ่มสินค้าในตะกร้าเสร็จสิ้น แสดงสรุปตะกร้าโดยวนลูปผ่านแผนที่ "cart" และพิมพ์ชื่อและราคาต่อหน่วยของสินค้าแต่ละรายการ
คำนวณจำนวนเงินรวมโดยรวมราคาของสินค้าทั้งหมดในตะกร้า
คำนวณจำนวนเงินภาษีโดยการคูณจำนวนเงินรวมด้วย 7% (เปอร์เซ็นต์ภาษี)
คำนวณราคารวมรวมภาษีโดยการเพิ่มจำนวนเงินรวมและจำนวนเงินภาษี
แสดงข้อความให้ผู้ใช้ป้อนรหัสส่วนลด (หากมี) หากผู้ใช้ป้อน "discount10" ให้ตั้งค่าเปอร์เซ็นต์ส่วนลดเป็น 3.0 หากผู้ใช้ป้อน "discount20" ให้ตั้งค่าเปอร์เซ็นต์ส่วนลดเป็น 5.0 หากผู้ใช้ป้อนค่าอื่นๆ หรือปล่อยให้ว่างเปล่าให้ตั้งค่าเปอร์เซ็นต์ส่วนลดเป็น 0.0
คำนวณจำนวนเงินส่วนลดโดยการคูณราคาสุทธิรวมภาษีด้วยเปอร์เซ็นต์ส่วนลด
คำนวณราคาสุทธิโดยการหักจำนวนเงินส่วนลดจากราคาสุทธิรวมภาษี
แสดงจำนวนเงินส่วนลดและราคาสุทธิพร้อมกับข้อความที่เหมาะสม
ให้ตรวจสอบให้แน่ใจว่าโปรแกรมจัดการข้อมูลที่ผู้ใช้ป้อนและทำการคำนวณได้อย่างถูกต้อง
